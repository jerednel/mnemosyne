## Method and scope

Coverage is **comprehensive by domain, not universal**: the Wikidata import
pulls scoped slices (software/technology companies, programming languages,
notable software, major platforms) with a per-slice notability floor
(Wikipedia sitelink count), rather than attempting a universal ontology.
Wikidata content is CC0; every imported entity records its QID in
`attributes.wikidata_qid` and its import file in seed provenance, so each
canonical fact traces to a public source.

Known gaps: people, locations, events, and non-tech industries are not yet
imported; descriptions-as-aliases (e.g. "the lakehouse vendor") exist only
in the curated base seed; slice membership uses direct `P31` typing, so
entities typed only via subclasses are missed. The overlay tier's
propose-and-review flow is the intended path for filling gaps from real
usage.
